Automation Testing với Selenium

1. Giới thiệu Selenium

15 Tháng 12, 2025

Selenium là bộ công cụ mã nguồn mở phổ biến để kiểm thử tự động trên trình duyệt web. Selenium hỗ trợ nhiều ngôn ngữ lập trình như Java, C#, Python, và nhiều trình duyệt khác nhau.

2. Các thành phần Selenium

  • Selenium WebDriver: Thực hiện các hành động trên trình duyệt như click, nhập dữ liệu, di chuyển chuột.
  • Selenium IDE: Công cụ ghi lại và phát lại test script nhanh chóng, phù hợp với người mới bắt đầu.
  • Selenium Grid: Chạy test song song trên nhiều môi trường và trình duyệt.

3. Ưu điểm Selenium

  • Hỗ trợ đa trình duyệt (Chrome, Firefox, Edge, Safari).
  • Dễ dàng tích hợp với CI/CD tools như Jenkins.
  • Hỗ trợ nhiều ngôn ngữ lập trình.
  • Mã nguồn mở, miễn phí sử dụng.

4. Cách viết test script cơ bản

Bắt đầu với Selenium WebDriver, bạn cần:

  1. Cài đặt Selenium WebDriver cho ngôn ngữ bạn chọn.
  2. Khởi tạo trình duyệt, mở trang web cần test.
  3. Xác định phần tử trên trang bằng XPath, CSS Selector.
  4. Thực hiện hành động: click, nhập dữ liệu, submit form.
  5. Đóng trình duyệt sau khi test xong.

5. Best Practices trong Selenium Testing

  • Sử dụng Page Object Model để quản lý element và test case dễ bảo trì.
  • Tích hợp với TestNG/JUnit để quản lý test case.
  • Sử dụng explicit wait để xử lý các phần tử động.
  • Chạy test song song để tiết kiệm thời gian.

6. Kết luận

Selenium là công cụ mạnh mẽ để tự động hóa kiểm thử web. Việc áp dụng Selenium giúp tiết kiệm thời gian, nâng cao độ chính xác và giảm rủi ro khi release sản phẩm.
